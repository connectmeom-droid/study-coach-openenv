# 📚 Study Coach OpenEnv

## Description
An AI environment that generates and optimizes study plans based on available time, weak subjects, and past performance.

## Tasks
- Task 1: Basic daily study planning  
- Task 2: Adaptive planning based on performance  
- Task 3: Long-term study optimization
